# Distributed Deadlock Detection Simulation

This project simulates distributed deadlock detection using a wait-for graph approach.

## Features
- Multiple sites with processes
- Resource allocation simulation
- Local wait-for graph generation
- Global wait-for graph construction
- Deadlock detection using cycle detection

## Files
- app.py → Streamlit application
- simulation.py → Simulation logic
- requirements.txt → Required libraries

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

## Output
- System state display
- Wait-for graphs
- Deadlock detection result

## Concepts
- Distributed systems
- Wait-for graph
- Cycle detection
